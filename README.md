# Nedabah Way — GitHub Pages Starter (초간단 웹사이트)

이 레포는 **“깃허브를 처음 쓰는 사람”**을 위해 만든 초간단 스타터입니다.

- ✅ 흰 배경 + 악센트 `#07930E`
- ✅ 설치 없이 바로 열리는 정적 웹사이트 (HTML/CSS/JS)
- ✅ GitHub Pages로 무료 배포 가능

---

## 1) 로컬에서 확인 (가장 쉬움)

1. 이 폴더를 다운로드
2. `index.html` 더블 클릭
3. 끝

---

## 2) GitHub에 올려서 “내 사이트”로 배포하기

### 방법 A) 제일 쉬운 방법 (GitHub Pages 설정)

1. GitHub에서 새 레포 만들기
2. 이 폴더의 파일을 업로드
3. 레포 설정(Settings) → Pages
4. **Branch: `main` / Folder: `/ (root)`**
5. 저장하면 주소가 생깁니다.

---

### 방법 B) 자동 배포 (GitHub Actions)

이 레포에는 `.github/workflows/pages.yml` 이 들어 있습니다.

- main 브랜치에 push 하면 자동으로 Pages 배포를 시도합니다.
- 만약 권한/설정이 막히면, GitHub 레포 Settings → Pages에서
  `Build and deployment` 옵션을 한번 확인하면 됩니다.

---

## 3) 수정 포인트 (딱 여기만 만지면 됨)

- **텍스트**: `index.html`
- **색/레이아웃**: `styles.css`
- **버튼 기능**: `app.js`
